# 📊 Laptop Productivity Analysis

Αυτό είναι το πρώτο μου Data Analysis Project! 🚀

Σκοπός του έργου ήταν να αναλύσω τη συμπεριφορά χρήσης του laptop μου, ώστε να κατανοήσω πώς κατανέμεται ο χρόνος μου και να βελτιώσω την παραγωγικότητά μου.

### ❓ Το Πρόβλημα
Ήθελα να απαντήσω σε ερωτήματα όπως:
* Ποιο ποσοστό του χρόνου μου καταναλώνεται σε διάβασμα και ποιο σε διασκέδαση;
* Ποιες δύο εφαρμογές χρησιμοποιώ περισσότερο;

### 🛠️ Μεθοδολογία & Εργαλεία
Η διαδικασία που ακολούθησα περιλαμβάνει:
1.  **Data Collection (Excel):** Καταγραφή δεδομένων σε πραγματικό χρόνο (Battery Level, Activity Type, Timestamp).
2.  **Data Cleaning & Transformation:** Έλεγχος για κενές τιμές και μορφοποίηση των δεδομένων για εισαγωγή στο Power BI.
3.  **Visualization (Power BI):** Σχεδιασμός διαδραστικού Dashboard για την οπτικοποίηση των αποτελεσμάτων.

### 💡 Βασικά Συμπεράσματα (Key Insights)
Από την ανάλυση των δεδομένων προέκυψαν τα εξής:
* **Κατανομή Χρόνου:** Το μεγαλύτερο μέρος του χρόνου μου αφιερώνεται στο Study, δείχνοντας ότι το laptop είναι κυρίως εργαλείο παραγωγικότητας.
* **Top Εφαρμογές:** Οι εφαρμογές με τη μεγαλύτερη κατανάλωση μπαταρίας είναι το **System** και ο **Brave Browser**.
    * *Ανάλυση:* Αυτό επιβεβαιώνει ότι η πλειοψηφία των δραστηριοτήτων μου (έρευνα, e-learning, studying) είναι web-based, καθιστώντας τον browser το σημαντικότερο εργαλείο μου.
* **Διαχείριση Ενέργειας:** Η υψηλή κατανάλωση του "System" υποδεικνύει σταθερή χρήση πόρων συστήματος, ενώ ο browser κυριαρχεί λόγω της multitasking φύσης της χρήσης μου.
